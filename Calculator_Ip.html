Volver

Las direcciones IP son únicas para cada máquina. Cada dirección es única para cada una de las interfaces de red IP de cada máquina.
Si una máquina dispone de más de una interfaz de red, necesitará una dirección IP para cada una.

Las direcciones IP tienen una longitud de 32 bits (4 bytes) los cuales representarían un conjunto de unos 4.500 millones de direcciones, útiles para ser asignadas a dispositivos de acceso a la red Internet.

La numeración en IP sigue una filosofía jerárquica. Cada dirección está formada por dos partes. Una corresponde a la red donde está la estación y la otra, a la propia estación.
